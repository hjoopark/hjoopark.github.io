Athena VS Macie

Author: Hyunjoo Park Date: June 30, 2021  23:40:22 Category: AWS-SAA

Athena

  • Athena는 대화형 쿼리 서비스이다
  • 표준 SQL을 사용하여 S3에 있는 데이터를 쿼리할 수 있다
  • Serverless
  • S3에 저장된 로그 데이터를 분석하는데 주로 사용한다

Athena가 사용되는 곳?

  • ELB 로그, S3접근 로그 등 S3에 저장된 로그 파일을 쿼리하는데 사용할 수 있다
  • S3에 저장된 데이터에 대한 비즈니스 보고서 생성
  • AWS비용과 사용 레포트 분석
  • 클릭 스트림 데이터에 대한 쿼리 실행

Macie

  • Macie는 AI를 사용하여 S3의 데이터를 분석하고 PII를 식별하는데 도움을 준다
  • 의심스러운 API 활동에 대한 CloudTrail 로그를 분석하는데 사용할 수도 있다
  • 대시보드, 보고서 및 경고 포함
  • PCI-DSS 규정 준수 및 ID 도용 방지에 적합
