AWS DataSync

Author: Hyunjoo Park Date: June 28, 2021  23:42:48 Category: AWS-SAA

DataSync 란?

AWS DataSync는 온프레미스 스토리지 시스템과 AWS 스토리지 서비스 간, 그리고 여러 AWS 스토리지 서비스 간의 많은 데이터 복사를 간소화, 자동화 및 가속화하는 온라인 데이터 전송 서비스이다.

DataSync에서는 특별 제작된 프로토콜을 사용하여 인터넷 또는 AWS Direct Connect를 통해 오픈 소스 도구보다 10배 더 빠른 속도로 안전하게 전송한다.

  • 온프레미스에서 AWS로 대량의 데이터를 전송할 수 있다
  • NFS와 SMB와 호환 가능한 파일시스템 사용
  • 매시간마다, 매일, 매주 복제할 수 있다
  • 복제를 시작하기 위해 DataSync 에이전트를 설치
  • EFS에서 EFS로 복제 가능하다
  • 일회성 데이터 마이그레이션이다
  • 동기화가 아니고 한번만 복제하는 경우

AWS DataSync 를 사용하는 이유?

AWS DataSync 를 사용하면 오픈소스 도구를 사용하여 사용자 지정 솔루션을 구축하거나 비싼 상용 네트워크 가속화 소프트웨어 라이선스를 사용 및 관리할 필요없이 수십억개의 파일이 포함된 대규모 데이터 세트를 복사할 수 있다.
